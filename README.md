# hex_to_variable
Mac command line program for converting RGB hex to variables in popular programming languages

Usage:

    hextovariable [-fX] [hex_string ...]
    
    Options [-f]:
      -fsi  swift ios
      -fsm  swift mac
      -foi  objc ios
      -fci  C# ios
    
    Hex String Formats:
      rgb - 3 Digit Hex Color
      rrggbb - 6 Digit Hex Color
      aarrggbb - 8 Digit Hex Color
